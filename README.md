# SimpleCommandTimer

Paper `1.16` - `1.21.x` ve Folia `1.21.1` ile uyumlu komut zamanlayıcısı.

## Build

```bash
mvn clean package
```

Jar çıktısı:

- `target/SimpleCommandTimer-1.0.0.jar`

## Komut

- `/sctimer reload`

## Config

`src/main/resources/config.yml` örnek formatı, gönderdiğiniz yapı ile uyumludur.
